# 🚚 delivery-team / monthly-reports

Интерактивный дашборд для отслеживания ключевых метрик и инициатив команды доставки по месяцам. Интерфейс выполнен в стиле GitHub.

---

## 📦 Содержимое

```
delivery-dashboard.html   — единый HTML-файл, не требует сборки или сервера
README.md                 — этот файл
```

## 🗂 Структура данных

Дашборд охватывает **5 периодов** в рамках **2 кварталов**:

| Период | Квартал |
|---|---|
| Октябрь 2025 | Q4 2025 |
| Ноябрь 2025 | Q4 2025 |
| Декабрь 2025 | Q4 2025 |
| Январь 2026 | Q1 2026 |
| Февраль 2026 | Q1 2026 |

Каждый период содержит данные по **5 блокам**:

| Блок | Ключевые метрики |
|---|---|
| 🚀 Высокая скорость доставки | 30-мин сервис (Сеть, Москва), среднее время доставки, охват зоны |
| ✅ Высокое качество доставки | Доля опозданий, доля обращений, потери за лояльность |
| 💰 Высокая экономическая эффективность | Стоимость доставки, % от выручки, заказов/час |
| 🛡️ Минимум рисков и штрафов | Инициативы по КИС АРТ, IoT, фотоконтролю |
| ⚙️ Стабильность IT продуктов | Минуты недоступности сервиса |

## 🧭 Навигация

### Левая панель (сайдбар)

- **Период** — выбор месяца. Переключает главную панель на обзор выбранного месяца.
- **Блоки** — открывает сводную страницу блока по всем месяцам.

> На мобильных устройствах сайдбар открывается через кнопку **☰** в левом верхнем углу.

### Вкладки (при просмотре месяца)

| Вкладка | Описание |
|---|---|
| 📊 Обзор | Карточки метрик с динамикой и целями квартала |
| 📋 Инициативы | Список инициатив по блокам, с поиском |
| 📈 Тренды | Графики динамики метрик по всем периодам |

### Страница блока

Открывается кнопкой **«Статистика блока»** рядом с заголовком раздела или через сайдбар → Блоки. Показывает на одной странице:

1. **История метрик** — таблица по всем 5 месяцам с цветовой индикацией
2. **Цели по кварталам** — карточки целей для каждого периода
3. **Инициативы по месяцам** — раскрывающиеся группы по периодам

## 🎨 Карточки метрик

Каждая карточка показывает:
- Текущее значение метрики
- Дельту к предыдущему периоду (▲ рост / ▼ снижение)
- Значение предыдущего периода
- Цель квартала (если задана)

Цветовая индикация карточки:
- 🟢 **Зелёная полоса** — метрика улучшилась
- 🔴 **Красная полоса** — метрика ухудшилась
- 🟡 **Жёлтая полоса** — без изменений

---

## 🔍 Поиск по инициативам

Строка поиска в верхней части страницы. Поиск работает по инициативам **текущего выбранного месяца**, подсвечивает совпадения и показывает счётчик результатов.

## ✏️ Как добавить новый месяц

1. Открыть `delivery-dashboard.html` в любом редакторе
2. Найти секцию `const DATA = {` в блоке `<script>`
3. Добавить новый объект по аналогии с существующими (например, `mar2026: { ... }`)
4. Добавить кнопку месяца в сайдбар (HTML) и в drawer (мобильный HTML)
5. Обновить массив `MONTH_ORDER` и данные для графиков `trendData`

### Структура объекта периода

```js
mar2026: {
  title: 'Март 2026',
  quarter: 'Q1 2026',
  metrics: {
    speed: [ /* массив метрик */ ],
    quality: [ /* ... */ ],
    economics: [ /* ... */ ],
    it: [ /* ... */ ],
  },
  targets: {
    speed: ['цель 1', 'цель 2'],
    quality: [ /* ... */ ],
    economics: [ /* ... */ ],
    risks: [ /* ... */ ],
  },
  initiatives: {
    '🚀 Высокая скорость доставки': ['инициатива 1', '...'],
    '✅ Высокое качество доставки': [ /* ... */ ],
    /* ... */
  }
}
```

### Структура объекта метрики

```js
{
  label: 'Название метрики',
  value: '3.6%',          // текущее значение
  prev: '3.25%',          // значение предыдущего периода
  prevLabel: 'февраль',   // подпись предыдущего периода
  status: 'green',        // 'green' | 'red' | 'yellow' | 'neutral'
  delta: '+0.35%',        // текст дельты
  better: true,           // true = улучшение, false = ухудшение, null = нейтрально
  target: '30%',          // цель (или null)
}
```
